## Product

**Pizza**

## Objective

The Product Configurator Tool enables customers to customize a pizza according to their preferences. Users can select the pizza size and toppings, view the final order, calculate the total price, and add the product to the cart.

---

## Flow Description

The pizza ordering system allows customers to configure a pizza based on their requirements through a simple step-by-step process.

### Flow Steps

1. Start
2. Select Pizza Size (Small, Medium, Large)
3. Choose Toppings (Veg, Chicken)
4. Display Final Order
5. Calculate Price
6. Add to Cart
7. End
